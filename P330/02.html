THE ALTERNATIVE CSS BOX MODEL

alternative CSS box model In the alternative box model, any width is the width of the visible box on the page. The content area width is that width minus the width for the padding and border. No need to add up the border and padding to get the real size of the box. 

        CSS box-sizing enables you to define how you
        want the height and width of an element to be 
        calculated.It means that you can use this
        property to specify whether the padding or border element      
        should be included in the height and width or not.


            The padding area is the space around the content 
            area and within the border-box.It can be applied
            to all sides of the box or to the specific,
            selected side(s) - top, right, bottom, 
            and/or left.
